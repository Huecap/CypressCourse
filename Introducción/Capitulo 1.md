## Instalación 

**Requisitos**
- NodeJS

**Pasos**
- Crear un directorio donde se desarrollara el proyecto
- Instalamos Cypress como una dependencia de desarrollo

```bash
npm install cypress --save-dev
```

- Ejecutamos cypress
```Bash
.\node_modules\.bin\cypress open
```
